El tiempo

La fuente de información que utiliza esta web para mostrar las condiciones actuales del tiempo y su pronóstico para los siguientes 4 días es de Open Weather Map Es una empresa de IT con experiencia práctica en Big Data y tecnologías geoespaciales. Utilizando su API para desarrolladores podemos mostrar los datos meteorológicos. Si en algún caso la previsión meteorológica no coincide con la realidad, disculpen las molestias.
